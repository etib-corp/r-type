Detailed Description

template<int WIDTH, int HEIGHT, typename T>
class LE::Matrix< WIDTH, HEIGHT, T >

A class for representing a matrix.

The Matrix class represents a matrix with the specified width and height.

Template Parameters
WIDTHThe width of the matrix.
HEIGHTThe height of the matrix.
TThe type of the elements in the matrix.
Note
The width and height must be greater than 0.
The type T must be an arithmetic type.
