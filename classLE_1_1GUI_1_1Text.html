Detailed Description

The Text class is responsible for managing text in the graphical system.

The Text class provides functionalities for loading and rendering text in the graphical system.

Constructs a new Text object (GUI).

This constructor initializes the Text object with the specified ID, font path, font height, and content.

Parameters
idThe ID of the text.
fontPathThe path to the font file.
fontHeightThe height of the font.
contentThe content of the text. (Default: "Hello World !")
fontWidthThe width of the font. (Default: 0)
Note
The fontHeight and fontWidth parameters are in pixels (px) and represent the size of the font.
