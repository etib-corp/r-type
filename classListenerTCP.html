Detailed Description

Manages TCP Listeners for the server.

This class is responsible for handling TCP Listeners using Boost.Asio. It provides methods to wait for incoming Listeners and to run the Listener handling loop.

◆ WaitForConnection()

void ListenerTCP::WaitForConnection ( )
overridevirtual

Asynchronously waits for incoming TCP Listeners.

This function initiates an asynchronous accept operation to wait for incoming TCP Listeners. When a Listener is established, it creates a new TCPSession, starts it, sends a welcome message, and stores the session. The function then recursively calls itself to continue waiting for new Listeners.

Note
This function uses Boost.Asio for asynchronous operations.
