Detailed Description

template<typename T>
requires std::is_arithmetic<T>::value
class LE::Vector3< T >

A 3D vector class template.

This class represents a 3D vector with x, y, and z components.

Template Parameters
TThe type of the components of the vector.
Note
The template parameter T must be an arithmetic type.
