Detailed Description

Manages scenes within the application.

The SceneManager class is responsible for managing the lifecycle of scenes, including playing, stopping, and switching between scenes. It maintains a collection of scenes and tracks the current and previous scenes.

Note
This class uses the SceneManagerError custom exception class to handle errors specific to scene management.
