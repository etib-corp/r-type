Detailed Description

Manages a UDP Listener for network communication.

The ListenerUDP class encapsulates the functionality required to manage a UDP Listener, including sending and receiving messages. It utilizes Boost.Asio for asynchronous network operations.

Note
This class is designed to be used in a server environment where multiple clients may connect and communicate via UDP.

The ListenerUDP class provides methods to start receiving messages, handle received messages, and send messages to a remote endpoint. It operates on a separate thread to ensure non-blocking network operations.
